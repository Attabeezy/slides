# 8-Day Intensive Python Course

Learn Python fundamentals in 8 focused days, from complete beginner to building real applications with data science tools.

## Course Overview

This intensive course condenses essential Python knowledge into 8 comprehensive days:

- **Days 1-5**: Core Python fundamentals
- **Days 6-7**: Advanced features and file operations
- **Day 8**: Object-Oriented Programming and next steps

After completing this course, continue with the Data Science modules included in this platform.

## What You'll Learn

- Python syntax and programming fundamentals
- All major data structures (lists, dictionaries, sets, tuples)
- Functions, modules, and code organization
- Object-oriented programming
- Error handling and file operations
- Web scraping basics
- Introduction to NumPy, Pandas, and data science

## Prerequisites

- Basic computer literacy
- No programming experience required

## Recommended Schedule

**Beginners**: 8 days (one day per module)  
**Programmers from other languages**: 4-5 days (combine related days)

## Continue Learning

After completing this 8-day course, continue with the **Data Science modules** available in this platform to master:

- NumPy for numerical computing
- Pandas for data manipulation
- Matplotlib for visualization
- Machine learning with Scikit-learn
- Deep learning with Keras
