# Python Installation

## What is Python?

Python is a high-level, interpreted programming language created by Guido van Rossum in 1991.

**Key features:**

- Easy to learn and read
- Interpreted (no compilation needed)
- Object-oriented
- Extensive standard library
- Cross-platform (Windows, macOS, Linux)

## Why Python?

Python is widely used in:

- Web development
- Data science and machine learning
- Automation and scripting
- Scientific computing
- System administration

# Installing Python

### Windows

1. Visit [python.org](https://www.python.org/)
2. Download the latest Python 3 installer
3. Run the installer
4. **Important:** Check "Add Python to PATH"
5. Click "Install Now"

## macOS

Python 3 may already be installed. To check:

```bash
python3 --version
```

If not installed, download from [python.org](https://www.python.org/) or use Homebrew:

```bash
brew install python3
```

### Linux

Most Linux distributions include Python. Install using your package manager:

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3

# Fedora
sudo dnf install python3
```

# Verifying Installation

Open your terminal or command prompt and run:

```bash
python --version
# or
python3 --version
```

You should see something like: `Python 3.11.0`
