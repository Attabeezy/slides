# Day 6 Exercises

## Level 1: Modules

1. Create a module with utility functions (sum, average, max, min)
2. Import and use your module in another file
3. Use math module to calculate circle circumference
4. Generate 10 random numbers between 1-100
5. Get current date and time using datetime

## Level 2: Higher-Order Functions

1. Use map() to square all numbers in a list
2. Use filter() to get only positive numbers
3. Use reduce() to find product of all numbers
4. Create a function that takes a function as parameter
5. Write a decorator that prints function name before execution

## Level 3: Advanced

1. Create a calculator module with all operations
2. Use list comprehension to create multiplication table
3. Flatten a nested list using comprehension
4. Create a dictionary of word frequencies
5. Write a decorator to measure function execution time

## Bonus

1. Create a module for text processing (count words, remove punctuation, etc.)
2. Use os module to list all Python files in a directory
3. Create a custom random password generator
4. Write decorators for logging and caching
