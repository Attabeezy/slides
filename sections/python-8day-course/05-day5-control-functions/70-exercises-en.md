# Day 5 Exercises

## Level 1: Conditionals

1. Check if a number is positive, negative, or zero
2. Check if a year is a leap year
3. Find the largest of three numbers
4. Check if a number is even or odd
5. Grade calculator (A: 90-100, B: 80-89, etc.)

## Level 2: Loops

1. Print numbers 1 to 10 using while loop
2. Sum of numbers 1 to 100
3. Print multiplication table (1-10)
4. Count vowels in a string
5. Find factorial of a number

## Level 3: Functions

1. Function to check if number is prime
2. Function to reverse a string
3. Function to find max in a list
4. Function to calculate circle area
5. Function with default parameters

## Bonus

1. Fibonacci sequence generator
2. FizzBuzz (print 1-100, "Fizz" for multiples of 3, "Buzz" for 5, "FizzBuzz" for both)
3. Password validator function
4. Calculator with functions for +, -, *, /
5. Function to find common elements in two lists
