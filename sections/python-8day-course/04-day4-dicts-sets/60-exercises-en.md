# Day 4 Exercises

## Level 1: Dictionary Basics

1. Create a dictionary for a person with name, age, and city
2. Access and print each value
3. Add a new key-value pair for 'email'
4. Update the 'age' value
5. Remove a key using pop()

## Level 2: Dictionary Operations

1. Create a dictionary of 5 fruits with their prices
2. Calculate the total price
3. Find the most expensive fruit
4. Create a new dictionary with fruits over $2
5. Merge two dictionaries using update()

## Level 3: Sets

1. Create two sets of numbers
2. Find union, intersection, and difference
3. Remove duplicates from a list using sets
4. Find common items between two lists
5. Check if one set is a subset of another

## Bonus

1. Count word frequency in a sentence using a dictionary
2. Create a phone book with add, search, and delete functions
3. Find unique letters in a string using sets
4. Group students by grade using nested dictionaries
